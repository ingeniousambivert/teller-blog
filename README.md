# Teller Blog

> A minimal blog


## Built with

**Web** :

- [React](https://www.reactjs.org)
- [NextJS](https://nextjs.org/)
- [Recoil](https://recoiljs.org/)

**Server** :

- [Strapi](https://strapi.io/)
- [NodeJS](https://nodejs.org)

**Database** :

- [MongoDB](https://www.mongodb.com)
- [Mongoose](https://mongoosejs.com/)
